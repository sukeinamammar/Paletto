Challenge 3 iOS Application

Paletto is a creative iOS application designed to let users play music with color. By interacting with a grid of colors, users can trigger sounds and explore music in a visual and intuitive way. The app focuses on creativity, accessibility, and inspiration through sensory interaction.

**Client**

AJ Simpson - Renaissance Program

This application was created as a client-based project for Challenge 3, designed to encourage creative expression through color-driven music interaction.

**Features**:

- Color-based sound playback

- Interactive grid UI

- TipKit onboarding and guidance

- Menu navigation

- Accessibility support

- Partial landscape mode support

**My Contributions**:

- Designed low-fidelity and high-fidelity UI sketches

- Implemented TipKit for in-app guidance

- Researched and explored the color blob visual element at the top of the screen

- Implemented menu and accessibility features

- Added partial support for landscape mode

- Collaborated with the team and adapted features as the app evolved toward the end of the project


**Future Improvements**:

- Complete and refine landscape mode

- Enhance accessibility features

- Expand sound customization and interactions

- Improve visual feedback and animations

**Goal**:

Paletto aims to inspire creativity by blending music and color into an intuitive, interactive experience.
